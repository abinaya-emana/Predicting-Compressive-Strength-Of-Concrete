# Predicting-Compressive-Strength-Of-Concrete

A machine-learning project that predicts the **compressive strength of concrete** based on its material composition.  
Includes model training, a clean Flask web interface, and Docker deployment.

# live project link : https://concrete-strength-prediction-ml4a.onrender.com/

---

##  What This Project Does
- Trains a Gradient Boosting Regressor model  
- Predicts concrete strength using 8 material features  
- Provides a web UI for input and results  
- Packs the entire app in Docker for easy deployment  

---

##  Tech Stack
- Python 3.10  
- Flask  
- scikit-learn  
- pandas, numpy  
- HTML + CSS  
- Docker
